## Icon 图标

### Demo
```tsx
/**
 * defaultShowCode: true
 */
import React from 'react';
import { WIcon } from 'w-react-ts';

export default () => <WIcon code="&#xe679;" />;
```

### API
|参数|说明|类型|默认值|
|--|--|--|--|
|code|图标编码|string【unicode】|-|
|className|类名|string|-|
|onClick|图标单击事件|function(e)|-|
